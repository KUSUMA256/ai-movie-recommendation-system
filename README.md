# AI Movie Recommendation System

This project is an AI-powered movie recommendation system built using Python, Flask, and Machine Learning.

## Features
- Movie recommendation based on similarity
- Trending movies dashboard
- Movie posters using TMDB API
- Search autocomplete
- Web interface using HTML, CSS, JavaScript

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS/JavaScript
- TMDB API

## Project Structure
backend/ – Flask API and recommendation logic  
frontend/ – Web interface  
dataset/ – Movie datasets  
models/ – Model files  
requirements.txt – Python dependencies

## How to Run
Install dependencies:

pip install -r requirements.txt

Run the backend:

python backend/app.py
